### Описание
Очень упрощенная версия Slack. Основные возможности системы:

  * Наличие каналов и управление ими (удаление, переименование)
  * Чат в каждом канале работающий в режиме реального времени

### Используемые технологии:

  * Ajax, REST
  * Websockets
  * React + Redux + Forms
  * Bootstrap on React
  * Webpack
  * Koa.js
  * Pug
